# A single-page app that fetches data from a free public API and lets the user search results, open a detailed view and save favourites (persisted with localStorage).

Anime Website Finder
A simple web application that uses AniList API to fetch and display trending and finsihed/Ongoing animes with title and aired time. Users can search animes and the result based on the input is displayed.

📖 Overview
A simple web application that uses AniList API to fetch and display trending and finsihed/Ongoing animes with title and aired time. Users can search animes and the result based on the input is displayed.


📝 Requirements
Responsive layout in Tailwind: header/hero, results grid, footer
Live search with loading, empty, and error states
Data fetched from your chosen API using async/await
A detail view (modal or separate view) for a selected item
Favourites saved and restored using localStorage
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6+)
Fetch API
AniList API

🔗 API 
AniList API

Endpoint:

POST 'https://graphql.anilist.co';
